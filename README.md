# 3d-web-avatar-viewer-threejs-react-threefiber
3D Web Avatar Viewer (Three.js, React Three Fiber, Ready Player Me/Mixamo rig)

## Project Setup

```
npx create-next-app@latest web-avatar-viewer
cd web-avatar-viewer
npm install three @react-three/fiber @react-three/drei three-stdlib
npm install zustand
```

Running the project: ```npm run dev```
